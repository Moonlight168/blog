## 4. 核心技术说明

### 4.1 Spring Boot

Spring Boot是一个基于Java的开源框架，用于创建独立的、生产级的Spring应用程序。它简化了Spring应用的配置和部署过程，通过自动配置和起步依赖使开发更加高效。

### 4.2 MyBatis-PLUS

MyBatis-Plus是MyBatis的增强工具，它在MyBatis的基础上提供了更多的功能和便利操作，如分页查询、批量操作、代码生成等。

### 4.3 MySQL

MySQL是一个开源的关系型数据库管理系统，具有高性能、可靠性和易用性特点。它适合作为Gupt Management System的主要数据存储。

### 4.4 Redis

Redis是一个开源的内存数据结构存储，用作数据库、缓存和消息代理。在系统中用于缓存热点数据、会话管理和临时数据存储。

### 4.5 Spring Security

Spring Security是一个功能强大且高度可定制的身份验证和访问控制框架。它是保护基于Spring的应用程序的标准。
